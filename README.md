# 🧠 NoBluff AI

**Stop Bluffing. Start Proving.**

NoBluff AI is a technical verification tool that analyzes your resume projects and simulates real interview scrutiny. It detects vague claims, identifies weak areas, and generates targeted technical questions to help you prepare with confidence.

---

## 🚀 Features

- 🔍 **Bluff Detection**  
  Identifies vague or weak project descriptions.

- ❓ **Interview Question Generation**  
  Produces realistic technical questions based on your projects.

- 🧠 **Confidence Scoring**  
  Evaluates how credible your project claims are.

- 📄 **Resume Parsing**  
  Extracts and analyzes project-level information.

- 💬 **Feedback System**  
  Helps improve answers and deepen understanding.

- ⚡ **Real-time Analysis UI**  
  Smooth, animated interface with step-by-step processing.
